# Backend (Express + Sequelize)

## Setup
1. Copy `.env.example` to `.env` and edit if needed.
2. Install packages:
   ```bash
   cd backend
   npm install
   ```
3. Run:
   ```bash
   npm run dev
   ```
This uses SQLite by default for easy local runs. To use MySQL/Postgres set DB_DIALECT and DB_URL in `.env`.
