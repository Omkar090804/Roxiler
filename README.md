FullStack Rating App
====================

This repo contains a minimal fullstack implementation (Express.js backend + React frontend) to satisfy the coding challenge requirements. The backend uses Sequelize + SQLite by default for easy local testing.

Important file: FullStack Intern Coding Challenge PDF specification (provided by user). See citation in chat: fileciteturn0file0

Run backend:
- cd backend
- copy .env.example to .env
- npm install
- npm run dev
- (optional) node src/seed.js to create sample admin/owner/store

Run frontend:
- cd frontend
- npm install
- npm run dev
